# AWS Student Builder Group at Georgia Tech

Official website for the AWS Student Builder Group at Georgia Tech (AWS SBG at GT) — a student-led, student-driven organization focused on learning about the cloud through AWS technologies.

## Structure

- `index.html` — single-page site (nav, hero, about, events, team, resources, join form, contact footer)
- `assets/` — logo and image assets

## Running locally

This is a static site with no build step. Serve the directory with any static file server, e.g.:

```bash
npx serve .
```

Then open the printed local URL in your browser.

## Sections

- **Home** — hero intro and key stats
- **About** — club mission and what members get
- **Events** — workshop, bootcamp, and hackathon formats
- **Team** — core team roles and open positions
- **Resources** — free AWS learning resources for members
- **Join Us** — membership interest form
- **Footer** — contact links (email, Instagram, LinkedIn, Meetup) and legal disclaimer
